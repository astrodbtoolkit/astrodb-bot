Unmatched Columns

  • plate, ifudsgn, versdrp2–versprim, platetyp, srvymode — Survey/version metadata. No AstroDB field. Use for filtering or store in Sources.comments.
  • ifuglon, ifuglat — Galactic coordinates. No AstroDB field for Galactic lon/lat.
  • nexp, exptime, drp3qual, bluesn2, redsn2, harname, frlplug, cartid, designid, cenra, cendec — Observation/instrument metadata. Use for filtering or skip.
  • airmsmin–transmax, mjdmin–mjdmax, gfwhm–zfwhm — Observing conditions. No AstroDB field.
  • mngtarg1–mngtarg3, catidnum, plttarg, manga_tileid — Target selection metadata. Use for row filtering.
  • ifudesignsize, ifutargetsize, ifudesignwrongsize — Instrument metadata. Skip.
  • zmin, zmax, szmin, szmax, ezmin, ezmax — Redshift ranges. Could go to ModeledParameters if needed.
  • probs, pweight, psweight, etc. — Survey weights. No AstroDB field.
  • nsa_field, nsa_run, nsa_camcol, nsa_version, nsa_nsaid, nsa_nsaid_v1b — NSA catalog metadata. nsa_nsaid could be Names.other_name if desired.
  • nsa_sersic_absmag, nsa_elpetro_absmag — Absolute magnitudes. Photometry stores apparent only. Options: ModeledParameters (parameter="absolute_magnitude_<band>", unit="mag") or skip.
  • nsa_elpetro_amivar — Magnitude inverse variance. Not direct photometry. Skip or ModeledParameters.
  • nsa_petro_flux, nsa_petro_flux_ivar, nsa_elpetro_flux, nsa_elpetro_flux_ivar, nsa_sersic_flux, nsa_sersic_flux_ivar — Petrosian/Sérsic fluxes. Not apparent magnitudes. ModeledParameters or skip.

Ingestion Notes

  • Source identifier: mangaid maps to Sources.source. Ensure each row gets a unique source value.
  • Duplicate columns: z and nsa_z both map to RadialVelocities.rv_kms — pick one. nsa_elpetro_ba and nsa_sersic_ba both map to Morphology.ellipticity — pick one or store both as separate ModeledParameters. nsa_elpetro_phi and nsa_sersic_phi both map to position_angle_deg — pick one. nsa_elpetro_th50_r, nsa_petro_th50, nsa_sersic_th50 all map to half_light_radius_arcmin — pick one or use ModeledParameters for extras.
  • Positions table: ifura/ifudec map to Positions. Each row needs a reference. Add SDSS/MaNGA reference to Publications first.
  • ModeledParameters: Requires model name and reference. Define a model (e.g. "NSA") and reference before ingestion.
